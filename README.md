# 大模型统一接口

各大厂的大语言模型非官方 SDK。主要特色是低依赖，支持以下模型：

- 百度千帆
- 智谱 Ai
- 腾讯混元
- 讯飞星火
- 商汤日日新
- MinMax
- 谷歌 Gemini
- 阿里通义千问

## 模型密钥申请

- 阿里 通义千问 <https://dashscope.console.aliyun.com/apiKey>
- 谷歌 Gemini <https://aistudio.google.com/app/apikey?hl=zh-cn>
- 讯飞 星火 <https://console.xfyun.cn/services/bm3>
